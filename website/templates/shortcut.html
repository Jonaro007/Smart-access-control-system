Apple Kurzbefehl Tutorial

Dieses Tutorial erklärt Schritt für Schritt, wie du deinen Apple Kurzbefehl einrichtest, um Daten sicher an unseren Server zu senden.

Schritt-für-Schritt Anleitung:

1. Kurzbefehl öffnen:

Öffne auf deinem iPhone die App Kurzbefehle und erstelle einen neuen Kurzbefehl.

2. Aktion "Inhalt von URL abrufen" hinzufügen:

Suche nach der Aktion Inhalt von URL abrufen und wähle sie aus. Stelle sicher, dass die Methode auf POST gesetzt ist.

3. URL eintragen:

Trage die URL ein, die auf deine Flask-Route zeigt, z.B.:
https://smart-lock.store/save_fingerprint

4. Formular-Daten hinzufügen:

Füge die folgenden Felder hinzu (Name muss exakt wie in Flask sein):

  • wert → Access-Code (geheim, schwer erratbar)
  • client_ip → Name oder Label, keine echte IP nötig
  • fingerprint → Eindeutiger String, z. B. UUID oder Hash
  • housenumber → Hausnummer, selbsterklärend

Input-Felder zum Kopieren

Geheim und stark, nicht erratbar.

Beliebiger Name, keine echte IP nötig.

Eindeutiger Fingerprint, schwer zu erraten.

Selbsterklärend.
